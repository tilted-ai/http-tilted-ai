About Us

Tilted AI is a collection of volunteer machine learning researchers[*] and engineers working to build and open source SOTA general AI models applicable to any problem with near-zero modification. Our focus is on the research and development of task agnostic architectures that enable users with limited machine learning experience to run and deploy these powerful models in a simple and intuitive manner.

Our process for the release of these architectures includes:

  1. Development
  2. Testing
  3. Release code on GitHub
  4. Publish architecture and experimental results on Arxiv

We also plan on releasing a command line tool for developers (both machine learning researchers and those unfamiliar with machine learning) to be able to quickly train and apply our models to their existing datasets.

We welcome anyone with the capacity to contribute.

* Our members have published architectural research and made contributions to open source reinforcement learning (RL) libraries. Notably, PyTorchRL, a scalable, open-source, RL library focused on modularity and simplicity and RLlib, an open source scalable RL library which is part of Ray, a universal API for building distributed applications.
